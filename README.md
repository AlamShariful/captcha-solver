# CAPTCHA Solver (Wordpress CAPTCHA Plug-In)

## Dependencies

1. Python 3
2. OpenCV 3 w/ Python extensions
3. numpy
4. sklearn
5. tensorflow
6. keras
7. imutils

## To run   
```
python3 extract_letters_from_captcha.py
python3 train_model.py
python3 solve_captchas.py
```
